# Boot Checklist

<!-- This runs when the gateway restarts. -->

1. Verify Telegram connection is active.
2. Check today's calendar for any events in the next 2 hours.
3. If reboot happened during waking hours, send a brief "I'm back online" message.
4. If reboot happened during quiet hours, stay silent — the morning briefing will cover it.
5. Check if any scheduled cron jobs were missed during downtime.
